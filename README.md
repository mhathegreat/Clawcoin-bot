## 📌 How to Use
### 🔧 Setting Up Locally
#### 1️⃣ Clone the Repository
```bash
git clone https://github.com/mhathegreat/CLAWCOIN-chatbot.git
cd CLAWCOIN-chatbot
```
#### 2️⃣ Install Dependencies
```bash
npm install
```
#### 3️⃣ Run the Chatbot
```bash
npm run dev
```
