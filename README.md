## 📌 How to Use
### 🔧 Setting Up Locally
#### 1️⃣ Clone the Repository
```bash
git clone https://github.com/mhathegreat/CLAWCOIN-bot.git
cd CLAWCOIN-bot
```
#### 2️⃣ Install Dependencies
```bash
npm install
```
#### 3️⃣ Run the Chatbot
```bash
npm run dev
```
